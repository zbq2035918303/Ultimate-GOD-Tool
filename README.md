## 🧠 项目介绍

Ultimate GOD Tool 是一个集成多种系统优化能力的工具：

- 📊 磁盘空间分析（Treemap 可视化）
- 🧹 一键清理（实时进度 + 日志）
- 📦 AI推荐迁移（安全软链接保护）
- ⚡ C盘空间优化（释放空间）
- 🎯 交互式操作（点击钻取）

---

## ✨ 核心功能

### 📊 磁盘分析
- WinDirStat 风格 Treemap
- 点击钻取目录
- 动画缩放
- 悬浮显示文件大小

---

### 📦 智能迁移
- AI推荐（仅推荐，不自动选择）
- 用户自主勾选
- 安全迁移（自动软链接）
- 支持回滚

---

### 🧹 一键清理
- temp / cache / recycle 清理
- 实时进度条
- 实时日志输出

---

### ⚡ C盘优化
- 关闭休眠释放空间
- 清理系统缓存
- 优化系统占用
