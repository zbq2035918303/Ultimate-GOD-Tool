## 🧠 核心功能

### 📊 磁盘分析
- WinDirStat 风格 Treemap
- 点击钻取目录
- 动画缩放
- 文件大小可视化

---

### 📦 智能迁移
- AI 推荐迁移（不自动操作）
- 用户自主选择
- 自动创建软链接（安全）
- 支持回滚

---

### 🧹 一键清理
- temp / cache / recycle 清理
- 实时进度条
- 实时日志输出

---

### ⚡ C盘优化
- 关闭休眠释放空间
- 清理系统缓存
- 系统占用优化
