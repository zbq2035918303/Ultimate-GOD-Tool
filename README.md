## 🧠 项目介绍

**Ultimate GOD Tool** 是一个集成多种系统优化能力的工具：

- 📊 磁盘空间分析（Treemap可视化）
- 🧹 一键垃圾清理（实时进度 + 日志）
- 📦 AI推荐文件迁移（安全软链接保护）
- ⚡ C盘空间优化（释放GB级空间）
- 🎯 交互式可视化（点击钻取 + 悬浮提示）

👉 类似：**WinDirStat + 腾讯电脑管家 + AI优化工具**

## ✨ 核心功能

### 📊 磁盘分析
- 彩色Treemap（类似WinDirStat）
- 点击钻取目录
- 动态缩放动画
- 鼠标悬浮显示文件大小

---

### 📦 智能迁移系统
- AI推荐（但不自动勾选）
- 用户自主选择
- 安全迁移（自动软链接）
- 支持回滚保护

---

### 🧹 一键清理
- 系统临时文件
- Windows缓存
- 用户缓存
- 实时进度条 + 日志

---

### ⚡ C盘空间优化
- 关闭休眠（释放数GB空间）
- 清理更新缓存
- 清空回收站
- 系统缓存优化
